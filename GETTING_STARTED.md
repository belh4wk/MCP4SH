# MCP4SH — Getting Started

Thanks for supporting MCP4SH ❤️  
This page covers install + activation in 2 minutes.

## 1) Download / Install
- Download the latest installer from the Releases page:
  - (link) **GitHub → Releases**
- Run the installer.
- Start SimHub.

## 2) Activate your license
1. Open **SimHub**
2. Go to **Plugins → MCP4SH**
3. Paste your license key
4. Click **Validate License** (or **Activate**)

If activation succeeds, you’ll see your tier (Supporter / Pioneer / Pro) in the plugin UI.

## 3) First-time setup (recommended)
- Start with the default profile
- Run one test session in a familiar car/track
- Only then tweak gain/intensity

## 4) Reinstall / new PC / hardware change
Your license has a machine limit. If you reinstall Windows or change hardware and hit an activation limit:
- Use the customer activation management (if enabled), or
- Contact support with your key and the email you purchased with.
